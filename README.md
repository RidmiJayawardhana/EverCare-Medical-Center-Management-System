# EverCare – Smart Healthcare Management Platform

EverCare is a modern healthcare management system designed to simplify medical service access and improve communication between patients and healthcare providers. The platform helps users easily find healthcare professionals, book appointments, manage medical services, and streamline healthcare operations through a user-friendly digital experience.

---

# 🚀 Features

## 👨‍⚕️ Patient Features

* User registration and secure login
* Search nearby doctors and healthcare professionals
* Book and manage appointments online
* View appointment history
* Access pharmacy and laboratory services
* Online payment processing
* Responsive dashboard interface

## 🏥 Healthcare Professional Features

* Doctor profile management
* Appointment scheduling and management
* Availability management
* Patient appointment tracking

## 👩‍💼 Receptionist Features

* Manage patient appointments
* Register patients
* Coordinate schedules
* Handle appointment records

## 💰 Accountant Features

* Payment tracking
* Invoice management
* Financial record handling
* Billing management

## 🧪 Additional Healthcare Services

* Find pharmacies nearby
* Find laboratories nearby
* Nutritionist support
* Caregiver services
* Midwife services
* Nurse appointment support
